# publimd

Publimd es una plataforma web innovadora para la creación y publicación de posts tipo blog. Su principal objetivo es permitir a los usuarios escribir contenido de forma rápida y sencilla utilizando Markdown (MD), mientras facilita la colaboración en equipo en tiempo real.

## Características Principales
- **Editor Markdown en tiempo real:** Escribe y visualiza tus posts de manera simultánea.
- **Trabajo Colaborativo:** Invita a otros autores y trabajen en el mismo documento de manera conjunta.
- **Lectura Optimizada:** Interfaz limpia y tipografía cuidada para la mejor experiencia de lectura.
- **SEO Dinámico:** Páginas estructuradas con metadatos de Schema.org para un excelente posicionamiento en buscadores.

## Tecnologías Utilizadas
Este proyecto (Frontend) está construido con las mejores herramientas del ecosistema moderno de JavaScript:
- **[Vue 3](https://vuejs.org/):** Framework progresivo de JavaScript con Composition API.
- **[Vite](https://vitejs.dev/):** Entorno de desarrollo ultrarrápido y empaquetador moderno.
- **[TypeScript](https://www.typescriptlang.org/):** Tipado estático para un código más robusto y escalable.
- **[Tailwind CSS (v4)](https://tailwindcss.com/):** Framework de utilidades CSS para un diseño ágil y responsivo.
- **[Pinia](https://pinia.vuejs.org/):** Gestión del estado de la aplicación.
- **[Vue Router](https://router.vuejs.org/):** Enrutamiento oficial para Vue.
- **[Marked](https://marked.js.org/):** Parser de Markdown a HTML rápido y completo con soporte de plugins (KaTeX, Mermaid).
- **[Highlight.js](https://highlightjs.org/):** Resaltado de sintaxis para bloques de código.

## Inicio Rápido

### Requisitos previos
- [Bun](https://bun.sh/) (Recomendado) o Node.js

### Instalación

1. Clona el repositorio e instala las dependencias:
```bash
bun install
```

2. Ejecuta el servidor de desarrollo:
```bash
bun run dev
```

3. Construye para producción:
```bash
bun run build
```
